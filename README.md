## Getting Started

Ensure you have >=go1.14.5

1. Clone the repo
    ```sh
    git clone https://github.com/expan75/sqlitego
    ```
2. Set gopath
3. Install dependencies
    ```sh
    go get
    ```
4. Run
    ```sh
    go run src/sqlitego
    ```

5. Check out available commands and go ham
    ```sqlitego
    sqlitego> help
    ```
